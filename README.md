# React Native Responsive UI Demo

A demo Expo app showcasing [@vincent-huy-uit/react-native-responsive-ui](https://www.npmjs.com/package/@vincent-huy-uit/react-native-responsive-ui).

## Getting Started

```bash
# Install dependencies
npm install

# Run on iOS
npm run ios

# Run on Android
npm run android
```
